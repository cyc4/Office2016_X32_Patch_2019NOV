### OPUTIL 自動安裝Office Patch

為加速你安裝Patch的速度，您可以使用以下工具進行自動Patch安裝。

下載路徑：https://github.com/cyc4/OPUtil

使用方式：把Oputil解壓縮後，放置與上述Patch同一個資料夾下

用Command Line進到Patch資料夾後輸入以下指令

cscript OpUtil.vbs /ApplyPatch

便會自動將Patch進行安裝。
